Intro: The game has been developed during 3 day session on Game Jam in Skopje (2019). We haven't been working on it since then and it's not perfected and optimized.
I was responsible for developing the game mechanics. 


Description: The game is about a dog that lost it's home. The dog will need to survive the everyday life on the streets and eventually find it's way home. It needs to eat, drink and increase happiness.
